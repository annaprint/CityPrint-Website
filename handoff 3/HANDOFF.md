# Сити Принт — Handoff for Claude Code

Интерактивный hi-fi прототип сайта типографии «Сити Принт» (Екатеринбург, с 2006 г.). Это единый HTML-файл на React + Babel standalone, с тремя переключаемыми арт-направлениями и тремя вариантами hero-секции.

---

## 1. Как запустить

Проект — статический. Ничего собирать не нужно.

```
# вариант 1 — просто открыть
open "Сити Принт.html"

# вариант 2 — поднять локальный сервер (нужен, если браузер ругается на CORS для React-скриптов)
python3 -m http.server 8000
# затем открыть http://localhost:8000/Сити%20Принт.html
```

Все зависимости (React 18, ReactDOM, Babel standalone) грузятся с unpkg через CDN с integrity-хэшами. Шрифты — Google Fonts (Fraunces, Manrope, Space Grotesk, JetBrains Mono). Фото — images.unsplash.com + одно клиентское фото в `assets/efremov-books.jpg`.

## 2. Структура

```
Сити Принт.html           ← главный файл: CSS, scripts, DOM-контейнер
src/
  data.jsx                ← данные (продукция, портфолио, отзывы, прайс, URL картинок)
  shared.jsx              ← Header, Footer, Placeholder, Marquee, Lightbox, useReveal, WordReveal, Logomark
  home.jsx                ← главная: HeroStats / HeroImage / HeroCalculator, Products, Portfolio, Why, Material, Reviews, CTA
  subpages.jsx            ← ProductPage, PortfolioPage, AboutPage, ProductsListPage
  app.jsx                 ← корневой App с роутером (localStorage-based), Tweaks-панель, лайтбокс
assets/
  logo-full.jpg           ← логотип клиента
  efremov-books.jpg       ← фото стопки альбомов Ефремова (от клиента)
handoff/
  screenshots/            ← референсные скриншоты всех стилей и страниц
  HANDOFF.md              ← этот файл
```

**Важно:** в проекте используется legacy-pattern для babel-компонентов — каждый `<script type="text/babel">` транспилируется в свою scope, поэтому в конце каждого JSX-файла есть `Object.assign(window, { ... })` для экспорта. Если будете рефакторить на ES-модули, сохраняйте этот протокол либо переезжайте целиком (с билдером).

## 3. Три стиля (Tweaks → «Арт-направление»)

Стиль переключается через `document.body.dataset.style`, вся визуальная разница — в CSS в `<style>` блоке главного файла. Значения:

| style      | Настроение | Типографика                      | Акцентная фишка |
|------------|------------|----------------------------------|-----------------|
| `atelier`  | Редакционный, премиум, по мотивам Imprimerie du Marais | Fraunces (serif) + Manrope | Крупные италики, мягкая палитра `#f4f1ec`, воздух |
| `presse`   | Производственный, крафт, печатный цех | Space Grotesk | Приводочные метки (REG ⊕ CMYK), жёсткие прямые углы, ступенчатые тени (`box-shadow: 6px 6px 0 var(--ink)`) |
| `riso`     | Игривый, ризограф | Fraunces + Space Grotesk | Точечные паттерны на фоне, mix-blend-mode: multiply на картинках, сдвиговые цветные тени |

Переключение пишется в `TWEAK_DEFAULTS` внутри `<script>` в HTML и сохраняется на диск хост-приложением (protocol `__edit_mode_set_keys`).

## 4. Три hero-варианта (Tweaks → «Hero»)

- `stats` — текст + плейсхолдер + 4 метрики под hero (20 лет / 70% / 5000+ заказов / доставка РФ)
- `image` — крупная фотография-обложка + короткий подзаголовок + 2 метрики
- `calc` — калькулятор стоимости рядом с заголовком: выбираешь тип продукции, тираж, страницы, бумагу, отделку → сразу видишь вилку цены

Прайс-лист иллюстративный, см. `PRICE_BASE` в `data.jsx`. До прода — привязать к реальному прайсу клиента.

## 5. Акценты (Tweaks → «Акцент»)

`red` (по умолчанию), `black`, `blue`, `green`. Меняется CSS-переменная `--accent`. Значение оттенка зависит от выбранного стиля (каждый стиль определяет свои оттенки).

## 8. Архитектурные заметки

### Роутинг

Клиентский роутер на `useState` + `localStorage` (ключ `citi-route`). Нет `history.pushState` — при перезагрузке страница остаётся та же, но URL всегда `/Сити Принт.html`. Для прода стоит перевести на честные URL (HashRouter или серверный роутинг на SSG).

```js
// src/app.jsx
navigate('product', 'books')  // { page: 'product', slug: 'books' }
navigate('portfolio')          // { page: 'portfolio' }
navigate('home')               // { page: 'home' }
```

### Лайтбокс портфолио

Общий компонент `Lightbox` живёт в `shared.jsx`, открывается через проп `openLightbox(index, items?)`. На главной работает со стандартным `PORTFOLIO`, на странице `PortfolioPage` — с расширенным списком (передаётся вторым аргументом).

### Reveal-анимации

`useReveal()` из `shared.jsx` — IntersectionObserver, добавляет класс `.in` когда элемент попадает в viewport. Класс `.reveal` задаёт начальное `opacity: 0; transform: translateY(16px)`.

### WordReveal

Каждое слово в hero-заголовке — отдельный `<span>` с задержкой появления. Italic-слова подсвечиваются акцентом и набраны display-шрифтом.

### Плейсхолдеры с картинками

`Placeholder` принимает `src` — если задан, показывает фото; иначе — диагональную штриховку (atelier), сетку (presse) или точки (riso). Это нужно чтобы в местах без фото всё равно была визуальная фактура.
