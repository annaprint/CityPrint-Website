## Навигация (шапка)

**Левая часть:** Логотип СИТИ ПРИНТ

**Центр:**
- Продукция (выпадающее меню: 8 продуктовых страниц)
- Портфолио
- О нас
- База знаний
- Контакты

**Правая часть:**
- Личный кабинет (ссылка на lk.citi-print.ru)
- Кнопка "Заказать" (красная, открывает форму)

**Мобильная:** бургер-меню + фиксированные кнопки звонка и Telegram внизу экрана.

## Футер

- Контакты: адрес, телефон, email, часы работы
- Навигация: продукция, сервис, информация
- Мессенджеры: Telegram, WhatsApp
- Юридическое: политика конфиденциальности, ИНН/ОГРН

## Портфолио

- Единая галерея всех работ с фильтрами по типу продукции
- Лайтбокс для просмотра фото
- 3-5 текстовых кейсов (задача клиента, что сделали, результат, фото)
- Подписи: что сделано, для какого клиента (с разрешения)

## Этапы запуска

1. **MVP:** Главная + 8 продуктовых + О компании + Контакты + Как заказать
2. **Фаза 2:** Портфолио с кейсами + База знаний + Техдокументация
3. **Фаза 3:** Интеграция калькулятора/ЛК через iframe
4. **Постоянно:** Наполнение базы знаний, обновление портфолио
