## Требования к фото

- Минимум 1200px по ширине (лучше 2400px для retina)
- Формат: JPG или PNG
- Без водяных знаков
- Хорошее освещение, без засветов
- Продукция на чистом фоне или в контексте (на столе, в руках) -- оба варианта полезны
- Горизонтальная ориентация предпочтительна для hero-блоков, квадратная -- для карточек
