# Спецификация: продуктовые лендинги Cityprint

**Дата:** 2026-04-25
**Статус:** утверждено пользователем
**Эталонный лендинг:** Печать каталогов и журналов

---

## 1. Цель

Превратить простые карточки продуктов (текущая ProductPage в handoff 3) в полноценные продуктовые лендинги, которые:
- конвертируют трафик из Яндекс.Директ / Google Ads в заявки;
- снимают страхи B2B-клиентов (сроки, качество, цена, ответственность);
- работают как страницы основного сайта с перелинковкой;
- оптимизированы под SEO и AI-агентов (AIO).

## 3. Дизайн-система

### Арт-направление

**Atelier как основа** с графическими приёмами из **Presse**:

- **Типографика:** Fraunces (serif, заголовки, цитаты) + Manrope (sans, body) + JetBrains Mono / Space Grotesk (моноширинный, лейблы, технические значения)
- **Палитра:** фон #f4f1ec (тёплый Atelier), текст #1a1a1a, вторичный #888, акцент #b8472c (фирменный красный)
- **Графика Presse:** приводочные метки (REG, крестики CMYK) как декоративные элементы в eyebrow-блоках, моноширинные лейблы на карточках, нумерация секций
- **Контрастные блоки:** тёмный фон (#1a1a1a) для блоков "Снятие рисков" и "CTA" — как MaterialSection и CTASection в прототипе
- **Пространство:** щедрые отступы, элементы "дышат" (принцип Atelier)
- **Углы:** мягкие border-radius 8-12px (Atelier), но лейблы и метки — жёсткие (Presse)

### Компоненты из прототипа (переиспользуются)

- `Header` / `Footer` / `MobileCTA` — без изменений
- `Placeholder` — для фото продукции
- `useReveal` — анимации появления при скролле
- `Lightbox` — для просмотра кейсов
- `WordReveal` — для hero-заголовков (опционально)

## 4. Структура лендинга (11 блоков + перелинковка)

### Блок 1: Hero (первый экран)

**Задача:** за 3-5 секунд объяснить продукт, аудиторию и CTA.

**Содержимое:**
- Eyebrow с названием категории + приводочная метка Presse
- H1: продукт + выгода (Georgia/Fraunces, крупный, тонкий)
- Подзаголовок: конкретика — тиражи, сроки, что включено
- Маркеры доверия: 3-4 коротких факта (моноширинный шрифт)
- 2 CTA-кнопки: "Рассчитать стоимость" (primary, pill) + "Отправить макет" (ghost, pill)
- Фото реального продукта справа (Placeholder с src)

**Макет:** 2 колонки (текст | фото), на мобильном — одна колонка, фото под текстом.

**Уникальность:** H1, подзаголовок, маркеры, фото — уникальны для каждого лендинга.

---

### Блок 2: Для каких задач (сценарии по сегментам)

**Задача:** клиент узнаёт свою ситуацию. Покрытие всех сегментов из маркетинговых стратегий.

**Содержимое:**
- Eyebrow: "01 — ЗАДАЧИ" + приводочная метка
- Заголовок: "С какими задачами приходят"
- Пилюли-сценарии (pill-shaped, белый фон, граница): каждый сценарий = реальная ситуация клиента из конкретного сегмента

**Макет:** flex-wrap контейнер с пилюлями.

**SEO/AIO:** каждый сценарий — потенциальный поисковый запрос.

**Уникальность:** полностью уникален для каждого лендинга.

---

### Блок 3: Что можно заказать (варианты продукта)

**Задача:** показать варианты без перегруза. Вести клиента за руку.

**Содержимое:**
- Eyebrow: "02 — ВАРИАНТЫ"
- Заголовок: "Что можно заказать"
- Карточки: сетка 3 колонки. Каждая карточка: моноширинный лейбл (Presse) + название + краткое описание
- Группы: по типу (скрепление, бумага, отделка, формат)

**Макет:** CSS Grid 3 колонки, на мобильном — 2 или 1.

**Уникальность:** полностью уникален. Это главный уникальный блок каждого лендинга:
- Каталоги: скрепление (скоба/КБС/пружина) + бумага + отделка
- Книги: переплёт (мягкий/твёрдый/пружина) + блок (ч-б/цвет) + бумага
- Упаковка: конструкции (коробка/обечайка/вкладыш) + материал + отделка
- Наклейки: материал (бумага/плёнка/прозрачная) + резка + защита
- Календари: тип (квартальный/настенный/настольный/перекидной)
- Блокноты: формат + обложка + крепление + внутренний блок
- Презентационные: тип (папка/буклет/брошюра/лифлет) + комплекты
- Выставки: состав комплекта (буклеты/бейджи/карты/номера)
- Пакеты: тип (бумажные/полиэтиленовые) + ручки + печать

---

### Блок 4: Почему это важно сделать правильно

**Задача:** эмоциональный якорь. Раскрыть боль клиента спокойно, без давления.

**Содержимое:**
- Eyebrow: "03 — ВАЖНО"
- Крупная цитата-боль (italic, Fraunces, 24px) — из маркетинговых стратегий
- Пояснительный текст (14px, спокойный тон)

**Макет:** одна колонка, максимальная ширина 600px, много воздуха.

**Уникальность:** текст уникален, структура шаблонная.

---

### Блок 5: Как Cityprint снимает риски

**Задача:** снять страхи. Обязательный блок на всех лендингах.

**Содержимое:**
- Eyebrow: "04 — ГАРАНТИИ"
- Заголовок: "Что мы берём на себя"
- 4-6 пунктов: красная вертикальная линия (Presse) + заголовок + описание
- Пункты: проверка макета, фиксированный срок, подбор материалов, контроль цвета, гарантия переделки, упаковка/доставка

**Макет:** тёмный фон (#1a1a1a), 2 колонки с пунктами.

**Уникальность:** шаблонный блок, одинаковый для всех лендингов (возможны незначительные вариации в описаниях).

---

### Блок 6: Примеры работ / кейсы

**Задача:** доказательства. Не галерея, а структурированные истории.

**Содержимое:**
- Eyebrow: "05 — КЕЙСЫ"
- Заголовок: "Примеры [продукта]"
- Карточки кейсов (2 колонки): фото + отрасль (моноширинный лейбл) + название + параметры (тираж/бумага/срок)
- Лайтбокс при клике (компонент из прототипа)

**Макет:** CSS Grid 2 колонки.

**Уникальность:** контент уникален, структура карточки шаблонная. Фильтрация кейсов по текущей категории продукта.

---

### Блок 7: Как проходит заказ (5 шагов)

**Задача:** снять ощущение "чёрного ящика".

**Содержимое:**
- Eyebrow: "06 — ПРОЦЕСС"
- Заголовок: "Как проходит заказ"
- 5 шагов горизонтально: номер в круге (красный, моноширинный) + название + описание
- Между шагами — пунктирная линия

**Шаги:**
1. Заявка — описываете задачу или присылаете макет
2. Проверка — менеджер проверяет файл и уточняет параметры
3. Расчёт — предлагаем 2-3 варианта по бюджету
4. Печать — производство с контролем цвета и качества
5. Доставка — упаковка и отправка

**Макет:** flex, горизонтальный. На мобильном — вертикальный.

**Уникальность:** шаблонный блок, одинаковый для всех лендингов.

---

### Блок 8: Что влияет на стоимость

**Задача:** прозрачность ценообразования + конверсия через калькулятор.

**Содержимое:**
- Eyebrow: "07 — СТОИМОСТЬ"
- Заголовок: "Что влияет на цену"
- Левая колонка: таблица факторов (тираж, полосы, бумага, отделка, срочность)
- Правая колонка: мини-калькулятор (переиспользуется из HeroCalculator прототипа, адаптированный под конкретный продукт)
- CTA под блоком: "Хотите точный расчёт? Отправьте параметры"

**Макет:** 2 колонки (факторы | калькулятор).

**Уникальность:** факторы частично уникальны, калькулятор параметризуется по продукту.

---

### Блок 9: Требования к макету

**Задача:** техническая информация + снятие страха для неопытных.

**Содержимое:**
- Eyebrow: "08 — МАКЕТ"
- Заголовок: "Требования к макету"
- 3 карточки с ключевыми параметрами: формат файла (PDF), цвет (CMYK), вылеты (3 мм) — крупные значения в моноширинном шрифте (Presse)
- Блок-подсказка: "Не знаете требований? Пришлите макет как есть — мы проверим бесплатно"
- Ссылка на подробную инструкцию (страница "База знаний")

**Макет:** 3 карточки в ряд + подсказка.

**Уникальность:** частично уникален (разные требования для разных продуктов).

---

### Блок 10: FAQ (аккордеон)

**Задача:** закрыть оставшиеся вопросы. SEO/AIO через FAQ Schema.

**Содержимое:**
- Eyebrow: "09 — ВОПРОСЫ"
- Заголовок: "Частые вопросы"
- 6-10 вопросов в формате аккордеона (клик раскрывает ответ)
- Вопросы подбираются под конкретный продукт из реальных запросов менеджерам

**Макет:** одна колонка, full-width, разделители.

**Разметка:** JSON-LD FAQPage Schema.org.

**Уникальность:** полностью уникален для каждого лендинга.

---

### Блок 11: Финальный CTA + форма

**Задача:** конвертировать. Не "контакты", а обещание результата.

**Содержимое:**
- Левая колонка: заголовок "Рассчитаем ваш [продукт]" + обещание + мессенджеры/телефон
- Правая колонка: короткая форма (4 поля): имя, телефон, что нужно, файл
- Кнопка "Отправить заявку"

**Макет:** тёмный фон (#1a1a1a), 2 колонки. Переиспользуется CTASection из прототипа (упрощённая до одного шага вместо трёх).

**Уникальность:** заголовок уникален, форма шаблонная.

---

### Перелинковка (между блоком 10 и 11)

**Задача:** удержание, SEO, кросс-селл.

**Содержимое:**
- Заголовок: "Также может быть полезно"
- 3 карточки смежных продуктов (название + подпись + ссылка)

**Подбор смежных продуктов:**
- Каталоги → книги, презентационные, упаковка
- Книги → каталоги, блокноты, наклейки (для авторов)
- Упаковка → наклейки, пакеты, каталоги
- Наклейки → упаковка, пакеты, визитки (приоритет 2)
- Календари → блокноты, презентационные, выставки
- Блокноты → календари, презентационные, выставки
- Презентационные → каталоги, выставки, календари
- Выставки → презентационные, наклейки, пакеты
- Пакеты → упаковка, наклейки, выставки

**Уникальность:** подбор уникален для каждого лендинга.

## 5. Подход к сегментации

Сегментация не через переключатели, а через **сценарии в блоке 2**. Каждый сценарий описывает реальную ситуацию клиента из конкретного сегмента:

| Сегмент | Пример сценария (каталоги) |
|---------|---------------------------|
| Марина (маркетолог) | Каталог продукции для отдела продаж |
| Дима (амбициозный) | Каталог с премиальной отделкой |
| Ирина (закупщик) | Регулярная печать каталогов по договору |
| Директор | Имиджевое издание компании |
| ИП | Первый каталог для малого бренда |
| Художник | Каталог выставки для галереи |

## 6. SEO / AIO требования

Для каждого лендинга подготовить:
- `<title>`: "[Продукт] в Екатеринбурге — Cityprint"
- `<meta description>`: 150-160 символов с ключевыми словами
- H1: один на странице, в hero
- H2: заголовки блоков
- FAQ Schema (JSON-LD FAQPage)
- Product Schema (JSON-LD)
- Alt-тексты для всех изображений
- Внутренние ссылки: перелинковка + хлебные крошки
- ЧПУ-адреса (см. раздел 2)

---

## 7. Адаптивность

- **Desktop:** 2-3 колонки в карточках, горизонтальный процесс
- **Tablet:** 2 колонки, процесс горизонтальный
- **Mobile:** 1 колонка, процесс вертикальный, CTA-кнопки на всю ширину, MobileCTA (звонок + Telegram) фиксирован внизу

---

## 8. Что НЕ входит в эту спецификацию

- Наполнение текстами (отдельный этап)
- Реальные фото продукции (нужна фотосъёмка)
- Подключение формы к Битрикс24 (отдельная задача)
- Лендинги приоритета 2 (визитки, плакаты, сертификаты, срочная печать, сегментные)
- Сегментные посадочные ("Для маркетологов", "Для авторов" и т.д.)
